# Prototype Scope Guard

## The Prototype Includes

- One arena map
- Destructible cover
- Up to 7 local players
- One enemy type
- One weapon
- Shared-lives co-op mode
- Instant respawn PvP time mode (kill-count scoring)
- Mid-match drop-in for new players (press A to join during gameplay)

---

## The Prototype Excludes

- Progression systems
- Unlockables
- Classes or builds
- Multiple weapons
- Bosses
- Procedural generation
- Story or lore
- Menus beyond basic join/start
- Online multiplayer
- Cosmetic customization
- Dash/dodge mechanic
- Stock/elimination PvP mode

---

## Rule

If a feature does not improve the core 2-second loop, it does not belong in the prototype.
